### Github

-  명령어
    - `git config —list`
        - 현재 config 설정된것 확인할 수 있음
    - `git config —unset 속성`
        - 설정한 속성 삭제
        - 글로벌로 설정된거는  `git config —unset —global 속성` 으로 삭제해줘야함
    - `git config —global user.name`
        - 글로벌 설정
        - [user.name](http://user.name) 과 [user.email](http://user.email) 등이 있다.
